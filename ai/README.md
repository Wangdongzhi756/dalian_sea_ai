# AI Project Docs

本目录保存大连海 AI SaaS 项目的规划、阶段任务和产品规格说明。

## 文档索引

- `project-plan.md`: 项目总体计划与里程碑。
- `current-plan.md`: 当前阶段执行计划。
- `long-term-plan.md`: 长期路线图。
- `spec.md`: 项目产品与技术规格。

## 当前代码基线

- 后端: `RuoYi-Vue`，已切换到 `springboot3` 分支，Spring Boot 3.x / JDK 17+。
- Web 管理端: `RuoYi-Vue3`，已切换到 `typescript` 分支，Vue 3 + TypeScript + Vite。
- 小程序/移动端: `RuoYi-App`，已切换到 `vue3` 分支，uni-app + Vue 3。

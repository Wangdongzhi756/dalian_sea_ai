# AI Project Docs

本目录保存大连海 AI SaaS 项目的执行计划和产品规格说明。

## 文档索引

- `project-plan.md`: 项目总体计划、阶段任务和开发路线。
- `current-plan.md`: 当前准备阶段执行清单。
- `spec.md`: 项目产品与技术规格。

## 当前代码基线

- 后端: `backend`，Spring Boot 3.x / JDK 17+。
- Web 管理端: `admin-web`，Vue 3 + TypeScript + Vite。
- 小程序/移动端: `mobile-app`，uni-app + Vue 3。

当前阶段优先清理脚手架痕迹，仅保留用户、组织、登录、权限、日志等安全与管理核心能力。

# 大连海 AI SaaS

大连海 AI SaaS 是一个三端一体项目，包含 Spring Boot 3 后端、Vue 3 + TypeScript Web 管理端，以及 uni-app + Vue 3 小程序/移动端。

## 目录

| 目录 | 说明 |
| --- | --- |
| `backend` | 后端服务，Spring Boot 3 / JDK 17+。 |
| `admin-web` | Web 管理端，Vue 3 + TypeScript + Vite。 |
| `mobile-app` | 小程序/移动端，uni-app + Vue 3。 |
| `ai` | 项目计划、当前计划和项目 spec。 |
| `agents.md` | AI 编码助手协作说明。 |

## 当前基线

- 后端: Spring Boot 3.x / JDK 17+。
- Web 管理端: Vue 3 / TypeScript / Vite / Element Plus。
- 小程序/移动端: uni-app / Vue 3。
